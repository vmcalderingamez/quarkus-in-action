# Repositorio de estudio: Quarkus in Action

Este repositorio es una guía práctica para comprender y aplicar los microservicios presentados en el libro *Quarkus en Acción*. Cada ejemplo del libro se implementa paso a paso, configurando y ejecutando los correspondientes servicios con Quarkus.

## Objetivo
- **Aprendizaje activo**: ejecutar y modificar cada ejemplo del libro.
- **Consolidar conceptos**: reforzar la teoría con código real.
- **Preparar despliegues**: preparar los microservicios para producción.

## Estructura
- `/service1`, `/service2`, … → proyectos individuales que representan los microservicios del libro.
- `/infra` → configuraciones comunes (docker, CI, pruebas, archivos complementarios).
- `/README.md` → este documento.
